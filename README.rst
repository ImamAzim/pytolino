template_mvc
============

this is a template for a python projects with a structure for mvc architecture (although one can use any architecture)

Look how easy it is to use:

    import project
    # Get your stuff done
    project.do_stuff()

Features
--------

- Be awesome
- Make things faster

Installation
------------

Install $project by running:

    install project
